# Исследование дальтонизма

## Описание

Исследовательский проект, посвященный изучению дальтонизма (цветовой слепоты). Проект включает материалы о типах дальтонизма, причинах возникновения, методах диагностики и современных подходах к коррекции.

## Что такое дальтонизм?

Дальтонизм (цветовая слепота) - это нарушение цветового зрения, при котором человек не может различать определенные цвета или их оттенки.

## Типы дальтонизма

### Протанопия (красная слепота)
- Неспособность различать красный цвет
- - Встречается примерно у 1% мужчин
 
  - ### Дейтеранопия (зеленая слепота)
  - - Неспособность различать зеленый цвет
    - - Встречается примерно у 1% мужчин
     
      - ### Тританопия (синяя слепота)
      - - Неспособность различать синий цвет
        - - Встречается реже (0,001%)
         
          - ### Ахроматопсия
          - - Полная цветовая слепота
            - - Самая редкая форма
             
              - ## Причины дальтонизма
